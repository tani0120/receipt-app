# AI会計システム ユーザー操作マニュアル

## Webアプリ (実務用ワークベンチ) の使い方

### 1. はじめに
このシステムは、お客様から受領した資料をAIが自動解析し、皆さんの「仕訳入力」を極限まで減らすためのツールです。
皆さんの主な役割は「入力」ではなく「**AIが作った仕訳の監査・承認**」になります。

---

### 2. 画面ごとの操作手順

#### Screen A: 顧問先一覧 (Clients)
**役割**: 担当する顧問先の状況を一目で確認します。

*   **[新着]バッジ**: 新しい資料が届いている顧問先には赤いバッジが付きます。
*   **操作**: 対象の顧問先列をクリックすると、ダッシュボード(Screen B)へ移動します。
*   **ステータス変更**: 「稼働中/停止中」の切り替えは管理者が行います。

#### Screen B: 業務進行管理 (Dashboard)
**役割**: その顧問先の「今月の作業進捗」を管理します。

*   **ステップ表示**:
    1.  **資料受領**: ファイルが届いているか。
    2.  **AI解析**: AI処理が終わっているか。
    3.  **1次仕訳**: 人間の確認待ち（Todo）は何件か。
*   **[入力開始]ボタン**: これを押すと、仕訳確認画面(Screen E)へ進みます。

#### Screen E: 仕訳確認・承認 (Workbench)
**役割**: 最も重要な画面です。AIが提案した仕訳をチェックし、確定させます。

1.  **画面構成**:
    *   **左側**: 領収書・請求書のプレビュー画像。
    *   **右側**: 仕訳データ（日付、科目、金額、摘要）。
2.  **操作フロー**:
    *   **確認**: 画像と仕訳を見比べます。
    *   **修正**: 間違っている箇所（例: 科目が「消耗品費」になっているが正しくは「雑費」）があれば、手入力で修正します。
    *   **承認**: 問題なければ「承認 (Approve)」ボタンを押します。次の画像へ自動的に進みます。
    *   **除外**: 「これは経費じゃない」等の場合、「除外 (Exclude)」ボタンを押してスキップします。
3.  **AI学習**:
    *   あなたが修正した内容はAIが学習し、次回から賢くなります。

#### Screen G: 会計ソフト変換 (Export)
**役割**: 確定したデータをCSVファイルとしてダウンロードします。

1.  **対象選択**: 出力したい顧問先を選びます。
2.  **ソフト指定**: 「弥生会計」「MFクラウド」など、顧問先に合わせた形式が自動選択されます。
3.  **ダウンロード**: ボタンを押すとCSVファイルが保存されます。これを会計ソフトにインポートしてください。

---

### 3. トラブルシューティング

**Q. AIが全く違う科目を提案してきます。**
A. Screen Eで正しい科目に修正して承認してください。AIはその修正パターンを学習し、次回以降の精度が向上します。

**Q. 「未払金」が勝手に入っています。**
A. クレカ払いや掛け払いと判定された場合、または支払い方法が不明な場合、システムは安全策として「未払金」等の標準決済科目をセットします。必要に応じて修正してください。

**Q. 口座の補助科目（○○銀行）が見つかりません。**
A. 新しい口座の場合、AIが「自動検知_...」といった仮の科目を作っている場合があります。それを使用するか、画面上の指示に従って正式名称を入力してください。
