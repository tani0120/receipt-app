## 2. マッチングルール

### 2-1. 勘定科目

- MF「各種設定 > 勘定科目」に登録済みの**名称と完全一致**
- 不一致の場合: インポート画面で「追加」or「変換」を選択可能
- 未選択で「次へ」→「未確定勘定」として取り込まれる（メモ欄に元の名称記録）
- **名称は30字まで**。31字以上は切り捨て

### 2-2. 税区分

- MF「各種設定 > 税区分」に登録済みの**「名称」と完全一致**
- **「省略名」ではマッチしない**（STREAMED help/8090で明記）
- 不一致の場合: 勘定科目と同様に「追加」or「変換」選択可能

### 2-3. インボイス区分

- MFで利用可能なインボイス区分の**名称と完全一致**
- 不一致の場合: インポート画面で変換画面が表示
- 空欄の場合: 取引日に基づきMFが自動で「適格」を適用

### 2-4. 補助科目・部門

- 勘定科目と同様に名称完全一致
- 補助科目名も30字まで

## 3. 複合仕訳の条件

| 条件 | 詳細 |
|------|------|
| 取引No | 同一の取引Noを共有 |
| 取引日 | 同一日付 |
| 貸借合計 | 借方合計 = 貸方合計 |

3条件を全て満たす行群が1つの複合仕訳としてインポートされる。

## 4. MF税区分マスター（エンジン対象範囲）

> 注意: MFの税区分マスターは非常に多い（簡易課税の事業区分別、個別対応方式の用途別など）。
> 本エンジンでは **STREAMEDの自動判定範囲と同等** の税区分のみを対象とする。

### 4-1. 仕入側（借方で使用）

| 税区分名称（MF正式名） | 税率 | 用途 |
|------------------------|------|------|
| 課対仕入10% | 10% | 標準税率の課税仕入 |
| 課対仕入8% | 8% | 旧税率8%の課税仕入 |
| 課対仕入(軽)8% | 8% | 軽減税率8%の課税仕入（飲食料品・新聞等） |
| 非課税仕入 | - | 保険料、土地の購入等 |
| 対象外 | - | 消費税の課税要件を満たさない取引 |

### 4-2. 売上側（貸方で使用）

| 税区分名称（MF正式名） | 税率 | 用途 |
|------------------------|------|------|
| 課税売上10% | 10% | 標準税率の課税売上 |
| 課税売上8% | 8% | 旧税率8%の課税売上 |
| 課税売上(軽)8% | 8% | 軽減税率8%の課税売上 |
| 非課税売上 | - | 保険金収入、土地売却等 |
| 対象外売上 | - | 消費税の課税要件外 |

### 4-3. インボイス区分

| インボイス区分 | 適用 |
|---------------|------|
| 適格 | 適格請求書発行事業者からの仕入（T番号あり） |
| 区分記載 | 適格請求書発行事業者**以外**からの仕入（経過措置対象） |
| （空欄） | MFが取引日に基づき「適格」を自動適用 |

> **経過措置の控除割合**（80%→70%→50%の段階的縮小）はMF側が取引日で自動判定する。
> エンジンは「適格 or 区分記載」のみ判定すればよい。

## 5. エンジンが出力すべきフィールド

`streamed_design_policy.md` のエンジン責務に基づき、CSV出力に必要な情報:

| フィールド | 対応CSV列 | エンジンの出力 |
|-----------|-----------|---------------|
| 取引日 | 取引日 | OCR抽出結果 |
| 勘定科目（借方） | 借方勘定科目 | MF科目名で出力 |
| 勘定科目（貸方） | 貸方勘定科目 | MF科目名で出力 |
| 税区分（借方） | 借方税区分 | MF税区分「名称」で出力 |
| 税区分（貸方） | 貸方税区分 | MF税区分「名称」で出力 |
| 金額（借方） | 借方金額 | **税込金額** |
| 金額（貸方） | 貸方金額 | **税込金額** |
| 摘要 | 摘要 | 発行者名 + 内容 |
| インボイス区分 | インボイス区分(借方/貸方) | 適格 or 区分記載 |
| 控除明細 | （複合仕訳の追加行） | 源泉徴収・振込手数料等 |

### 注意事項

- **税額列（借方税額・貸方税額）はエンジンから出力しない**（空欄）。MFのユーザー設定（税込/税抜経理）に応じてMFが自動計算する。
- **MF仕訳タイプ**は「インポート」固定で良い。
- **決算整理仕訳**は空欄（通常仕訳のみ対象）。
